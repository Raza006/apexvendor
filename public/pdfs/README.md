## PDF Content Suggestions

Each PDF should include:
- **Product Name** and description
- **Step-by-step instructions** on how to access the vendor
- **Vendor contact information** (if applicable)
- **Your support contact** information
- **Branding** (logo, colors, company name)
- **Important notes** or tips for using the vendor

## File Requirements

- **Format**: PDF (.pdf)
- **Max File Size**: 10MB per file (recommended under 5MB)
- **Naming**: Exact match to the names above (case-sensitive on some systems)

## Testing

After uploading PDFs:
1. Deploy your site to Vercel
2. Make a test purchase
3. Check that the PDF is attached to the email
4. Verify the PDF opens correctly on both mobile and desktop

---

**Note**: If a PDF file is missing, the email will still be sent but without an attachment. Check your Vercel logs for warnings about missing PDF files.
